El bot cuenta con una lista de comandos finita que a continuación se decriben:
•	$ version: Muestra la versión de la aplicación
•	!@wheramI : Muestra en que directorio está posicionado el script
•	!@showFiles directorio : Recibe como argumento un directorio, sin argumentos lista el contenido del directorio actual
•	!@cifraBinXor archivo : Ejecuta la operación XOR en cada byte del archivo
•	!@cifraBinRot archivo: Realiza una rotación de 180 grados al archivo en modo binario.
